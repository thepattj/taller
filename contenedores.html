Estilo CSS

las aplicaciones se dividen en tres capas: la de estructura y cotnenido (HTML), la capa visual(CSS) y la capa funcional (JS). La capavisual es la que se encarga de acomodar, colorear, agrupar y formatear lso distintos elementos de HTML, sin ella los contenedores no tiene absolutamente ninguna funcionalidad, mas que la de agrupar contenidos. Este lenguaje (Cascade Style Sheet) se puede escribir dentro de las etiquetas, dentro del documento(en el bloque de etiquetas STYLE) o en un documento externo (.CSS). Cada unad e estas formas tien un ambito y orden de importancia diferente.

Por ejemplo, cuando escribimos dentro de una etiqueta los estilos a traves del artibuto STYLE, el alcance que tiene es el minimo, ya que solo modifica a esa etiqueta en espeficio, en cambio, tiene la importancia mas alta (o la ultima palabra). cuando escribimos CSS dentro de un bloque de etiquetas STYLE se vuelve su alcance un poco mas grande, ya que afecta a todoas las etiquetas que se modifican en ese documento, y tiene un nivel de importancia media. El ultimo es por medio de un documento por separado, es es la manera ideal de escribir CSS, ya que podemos afectar a N cantidad de paginas al mismo tiempo desde el mismo documento

la Sintaxis de CSS es bastante sencilla. Primero que nada, necesitamos establecer un "SELECTOR". Los selectores son los que inidican que elemento se va a seleccionar para que sus propiedades sean modificacas. Existen tres tipos de selectores: los de etiqueta, identificador y clases. Los selectores de etiqueta modifican a todas las etiquetas que coincidan con el selector.

//SELECTOR DE ETIQUETA
p{
color:red;
}

Como se puede apreciar en el codigo anterior, la estructura de un bloque de CSS es la siguiente: comenzamos por el selector, abrimos las llaves, dentro de ellas colocamos la propiedad que se modificara segida de dos puntos(:) y posteriormente el nuevo valor. Todas las lineas de CSS terminan con punto y coma (;).

Los otros dos selectores (ID y CLASS) funcionan por medio de atributos con el mismo nombre. Los ID tiene la caracteristica de que deberia de ser unicos por pagina, cuando escribimos este selector en CSS debemos colocar el signo (#) antes del nombre del ID. Las clases tiene diferentes caracterisiticas, comenzando por que el atributo que se coloca es el atributo CLASS, estas pueden repetirse la cantidad de veces que sea necesario en un documento, tambien pueden ser acumulablres, es decir que un elemento puede tener mas de una clases (separada por espacios). Podriamos definir las clases como las "caracteristicas" de un elemento. Las clases comienzan con un punto(.)

#identificador{
propiedad:valor;
}

.clase{
propiedad:valor;
}

Existen diferentes tipos de propiedades de CSS, todas tienen un significado en ingles y son bastante descriptivas. Existen algunas propiedades en especifico que puden ser generales y especificas, por ejemplo el margen(MARGIN) que podemos colocar en una sola propiedad o alguna mas especifica(MARGIN-LEFTT)
